Structure of the MoMA Project Website

The Dataset page features information on the dataset we decided to use for this project as well as a sample of the dataset.

The Algorithm page explains how we designed an algorithm to analyse the gender distribution of Swedish artists in the MoMA collection. It outlines the key steps involved in creating the algorithm and our reasons for using pseudocode.

The People page includes short biographies of each group member and the ways they contributed to the MoMA Dataset Project.

Each page also includes a painting by a Swedish artist from our dataset.

Created for educational purposes as an assignment for the course "Datalogical Thinking" (Master's Programme in Information Science: Digital Environments, University of Borås) by the students: Dana Neumannová, Isabelle Brandt, Julia Milder and Orestis Georgalas.
